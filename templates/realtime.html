本機執行步驟
1
確認已安裝 Python 與相依套件：
pip install -r requirements.txt
2
根據訓練項目，在終端機執行對應腳本：
python Long.py        # 正手長球
python Longbackhand.py  # 反手長球
python ForeShort.py    # 正手小球
python BackShort.py    # 反手小球
python ForeHandFlick.py # 正手挑球
python BackHandFlick.py # 反手挑球
3
攝影機視窗會自動開啟，站在攝影機前開始練習即可
